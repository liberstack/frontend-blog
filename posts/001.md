# Fundamentos da Web

Antes de escrever uma linha de JavaScript, é importante entender o ambiente onde ele roda: o navegador.

---

## Como o navegador funciona

Quando você digita um endereço e pressiona Enter, o navegador faz uma requisição a um servidor e recebe três tipos de arquivo:

- **HTML** — a estrutura da página
- **CSS** — a aparência
- **JavaScript** — o comportamento

Esses três arquivos são interpretados juntos. O HTML é lido primeiro, o CSS é aplicado em seguida, e o JavaScript é executado depois que a página está carregada.

---

## O que é o DOM

DOM significa *Document Object Model*. É a representação da página HTML como uma árvore de objetos que o JavaScript pode ler e modificar.

Quando o navegador lê seu HTML:

```html
<h1 id="titulo">Olá</h1>
<p>Primeiro parágrafo.</p>
```

Ele cria internamente uma estrutura como esta:

```
document
└── html
    └── body
        ├── h1#titulo
        └── p
```

O JavaScript acessa essa árvore para alterar conteúdo, adicionar elementos, responder a cliques e muito mais.

---

## Seu primeiro script no navegador

Crie um arquivo `index.html` com o seguinte conteúdo:

```html
<!DOCTYPE html>
<html lang="pt-BR">
<head>
  <meta charset="UTF-8" />
  <title>Primeiro script</title>
</head>
<body>
  <h1 id="titulo">Olá</h1>
  <script src="app.js"></script>
</body>
</html>
```

Crie o arquivo `app.js` na mesma pasta:

```js
const titulo = document.getElementById("titulo");
titulo.textContent = "JavaScript está funcionando.";
```

Abra o `index.html` no navegador. O texto do `<h1>` será alterado pelo JavaScript antes de você ver a página.

---

## O que aconteceu

1. O navegador leu o HTML e criou o DOM
2. Encontrou a tag `<script>` e carregou o `app.js`
3. O JavaScript selecionou o elemento pelo `id`
4. Alterou o texto do elemento

Esse ciclo — selecionar, ler, modificar — é a base de tudo que você vai fazer com JavaScript no browser.
