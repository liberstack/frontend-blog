# JavaScript no Browser

Com o DOM acessível, o próximo passo é reagir ao que o usuário faz: cliques, digitação, envio de formulário.

---

## Selecionando elementos

O método mais usado para selecionar elementos é o `querySelector`. Ele aceita qualquer seletor CSS:

```js
document.querySelector("#titulo");       // por id
document.querySelector(".card");         // por classe
document.querySelector("button");        // por tag
```

Para selecionar múltiplos elementos:

```js
document.querySelectorAll(".item");      // retorna uma NodeList
```

---

## Ouvindo eventos com addEventListener

Para reagir a uma ação do usuário, você registra um *event listener* no elemento:

```js
const botao = document.querySelector("#meu-botao");

botao.addEventListener("click", function () {
  console.log("botão clicado");
});
```

Você pode usar arrow function também:

```js
botao.addEventListener("click", () => {
  console.log("botão clicado");
});
```

---

## Alterando o conteúdo da página

Depois de selecionar um elemento, você pode modificá-lo:

```js
// Alterar texto
elemento.textContent = "novo texto";

// Alterar HTML interno
elemento.innerHTML = "<strong>negrito</strong>";

// Alterar estilo
elemento.style.color = "red";

// Adicionar ou remover classe
elemento.classList.add("ativo");
elemento.classList.remove("ativo");
elemento.classList.toggle("ativo");
```

## Exemplo prático: contador de cliques

```html
<!DOCTYPE html>
<html lang="pt-BR">
<head>
  <meta charset="UTF-8" />
  <title>Contador</title>
</head>
<body>
  <p id="contagem">0</p>
  <button id="btn">Clique aqui</button>
  <script src="app.js"></script>
</body>
</html>
```

```js
let count = 0;

const contagem = document.querySelector("#contagem");
const btn = document.querySelector("#btn");

btn.addEventListener("click", () => {
  count += 1;
  contagem.textContent = count;
});
```

Cada clique incrementa a variável e atualiza o texto na tela. Sem reload, sem backend.

---

## Eventos mais usados

| Evento | Quando dispara |
|---|---|
| `click` | clique do mouse |
| `input` | digitação em um campo |
| `change` | alteração e saída do campo |
| `submit` | envio de formulário |
| `keydown` | tecla pressionada |
| `DOMContentLoaded` | DOM pronto |
