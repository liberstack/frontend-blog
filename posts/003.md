# Dados e Comunicação

A maioria das aplicações web precisa buscar dados de alguma fonte externa. No browser, isso é feito com a função `fetch()`.

---

## O que é uma API

API (*Application Programming Interface*) é um endereço que retorna dados, geralmente no formato JSON.

Exemplo de resposta JSON:

```json
{
  "nome": "Guilherme",
  "linguagem": "JavaScript",
  "nivel": 3
}
```

JSON é texto estruturado. O JavaScript converte esse texto em objeto usando `JSON.parse()`, ou automaticamente com `.json()` na resposta do fetch.

---

## Como funciona o fetch

`fetch()` faz uma requisição HTTP e retorna uma *Promise* — uma operação assíncrona que vai completar no futuro.

A forma mais legível de trabalhar com Promises é usando `async/await`:

```js
async function buscarDados() {
  const resposta = await fetch("https://api.exemplo.com/dados");
  const dados = await resposta.json();
  console.log(dados);
}

buscarDados();
```

- `await fetch(url)` — espera a requisição terminar e retorna a resposta
- `await resposta.json()` — converte o corpo da resposta de JSON para objeto JS

---

## Exemplo prático: consumindo uma API pública

A API do GitHub retorna dados de qualquer usuário publicamente:

```js
async function buscarUsuario(username) {
  const res = await fetch(`https://api.github.com/users/${username}`);
  const user = await res.json();

  document.querySelector("#nome").textContent = user.name;
  document.querySelector("#repos").textContent = user.public_repos;
}

buscarUsuario("liberstack");
```

HTML correspondente:

```html
<p id="nome"></p>
<p id="repos"></p>
```

## Tratando erros

Requisições podem falhar. Use `try/catch` para lidar com isso:

```js
async function buscarDados() {
  try {
    const res = await fetch("https://api.exemplo.com/dados");
    if (!res.ok) throw new Error(`Erro: ${res.status}`);
    const dados = await res.json();
    renderizar(dados);
  } catch (e) {
    console.error("Falha na requisição:", e.message);
  }
}
```

`res.ok` é `true` quando o status HTTP é entre 200 e 299.

---

## Carregando dados locais

`fetch()` também funciona com arquivos locais do seu projeto:

```js
const res = await fetch("./data/posts.json");
const posts = await res.json();
```

Isso é exatamente o que blogs e SPAs estáticos fazem — sem banco de dados, sem backend.
